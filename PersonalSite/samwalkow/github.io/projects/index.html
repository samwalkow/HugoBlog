Domain Stories - Astronomy

January 2019 - In Progress

This project is part of a larger effort to understand users mental model of data, and how that is applied when using scientific software to answer research questions. This understanding is an initial step to expanding domain specific software (in this case, yt) beyond computational astrophysics.

Understanding user's thought process and software interaction will start with their description and narrative of how they conduct analysis, step by step. Users draw their process, and annotate it with code and the reasons why they took that step, and what the goal was.

I captured three domain stories of users implementing yt for astrophysics analysis and presented my results at the 2019 What is Information Conference (virtually) and at the 2019 RHytHM Conference (a yt community event).
